2. Класс Tasks1_5
Содержит статические методы для решения задач 1-5.
Методы:
ReplaceNeighborsIfNotEqual(LinkedList<object> list, object elementE, object elementF)
Назначение: Заменяет соседей элемента E на элемент F, если соседи не равны между собой

Алгоритм:
Проход по всем узлам связанного списка
Поиск элементов, равных E
Проверка наличия обоих соседей
Сравнение соседних элементов
Замена на F при неравенстве соседей

AnalyzeCultur(HashSet<string> allBooks, HashSet<HashSet<string>> readersBooks)
Назначение: Анализ сельскохозяйственных культур в кооперативах

AnalyzeCommonCharactersInWords(string filePath)
Назначение: Определение символов, присутствующих в каждом слове текстового файла

Алгоритм:
Чтение текста из файла
Разделение на слова
Создание множеств символов для каждого слова
Нахождение пересечения всех множеств

FindSchoolsAboveDistrictAverage(string inputFilePath)
Назначение: Определение школ со средним баллом выше среднего по району

Алгоритм:
Чтение данных об учениках (Фамилия, Имя, Школа, Балл)
Расчет среднего балла по району
Расчет средних баллов для каждой школы
Выявление школ с показателем выше районного
Форматированный вывод результатов
